Prostephanus model

The model originally published in Meikle et al. (1998))was re-implemented in modern BoxScript. You can find the boxscripts below ordered by the original figure numbers. You can run the boxscripts yourself. For that purpose you must install Universal Simulator. Then, for example, to run the fig_2_dev_rate.box⏷ script, you write at the Universal Simulator prompt:

> run models/prostephanus/fig_2_dev_rate.box

Then paste the clipboard at the R prompt to see the output. A full description of how to handle and run boxscripts are found in Running simulations.

The cmbp plugin

The aphid plugin contains the model building blocks needed to run the cereal aphid-fungus model. The functionality of the building blocks (classes) is described below

You can run the boxscripts shown in the sub-sections below yourself. For that purpose you must install Universal Simulator. For example, to run the aphid_juvenile_survival.box⏷ script, you write at the Universal Simulator prompt:

> run models/aphid/aphid_juvenile_survival.box

Paste the clipboard at the R prompt to see the output as described in Running simulations.
